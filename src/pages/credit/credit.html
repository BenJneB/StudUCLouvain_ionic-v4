UCLCampus est une application créée par des étudiants pour des étudiants. Son but est d'accompagner les étudiants de l'UCL dans leur vie quotidienne sur le campus.

Sa création est inscrite comme sujet d'un mémoire réalisé par Marchesini Bruno & Daubry Benjamin supervisé par leur promoteur Deville Yves.

Toute contribution pour l'ajout de nouvelles fonctionnalités est la bienvenue. Ce projet étant open-source, si vous voulez participer à son développement vous pouvez trouver le github public via ce lien: https://github.com/bmarchesini/UCLCampus

Pour toute information supplémentaire, il est possible de nous contacter sur les adresses mails suivantes: bruno.marchesini@student.uclouvain.be benjamin.daubry@student.uclouvain.be

Nous remercions Jérôme Lemaire et Corentin Lamy pour avoir commencé son développement durant l'année 2016-2017.
